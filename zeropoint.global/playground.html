Governance Playground

Type an action. Watch ZeroPoint evaluate it in real time — Guard, Policy, and Audit, all running on live Rust infrastructure.

This playground connects to a live ZeroPoint server. No simulation. No mocks. Every evaluation runs through the same GovernanceGate that governs production deployments.

Audit Trail

Every governance decision is hash-chained into an immutable record. If any entry is modified or deleted, verification detects the break.

Server Identity

Every ZeroPoint node generates a unique Ed25519 keypair. This identity signs audit entries, addresses the node in mesh networks, and anchors trust tier verification. If the backend were swapped, this identity would change.
