ZeroPoint Sentinel

Governance at the network edge. DNS filtering, device monitoring, anomaly detection — with cryptographic mesh participation.

# Install on any supported platform (auto-detects)

What Sentinel Does

⊘
DNS Filtering
Block ads, trackers, and malware at the DNS level using Steven Black blocklists. Every query governed by policy.
📡
Device Monitoring
Track every device on your network via DHCP. Block rogue MACs. Detect unauthorized connections.
◉
Anomaly Detection
DNS spikes, port scans, DGA domains, device floods. Detected in real time with configurable thresholds.
📋
Audit Trail
Every policy decision recorded in a Blake3 hash-chained SQLite ledger. Tamper-evident by design.
🔔
Alert Notifications
Push to Ntfy, Slack, or any webhook. Critical alerts repeat until acknowledged. Configurable risk thresholds.
◈
Mesh Participation
Ed25519 identity, AgentAnnounce protocol, heartbeat. A first-class peer in the ZeroPoint trust mesh.
