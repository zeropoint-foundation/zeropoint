Getting Started

ZeroPoint is open-source. You don't need permission to start. Clone the repo, run cargo build, and follow any track. Both courses are self-contained and work with a local instance or against any ZeroPoint server endpoint.

Prerequisites: Basic familiarity with concepts like hashing, digital signatures, and API design helps but isn't required. The courses build up from first principles.
